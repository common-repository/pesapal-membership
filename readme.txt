Pesapal Membership is a membership management plugin using pesapal

== Description ==

Create memberships on your website and bill users using PesaPal https://www.pesapal.com/ . Once confirmation of the transaction is returned using the IPN URL, the user is emailed with their login credentials. This is a basic plugin of PesaPal membership and we will be releasing more on www.shumipress.com 

Main Features:

* PesaPal Integration
* Payment via PesaPal gateway


There are still many more features we are working on that are yet to come

== Installation ==

1. Upload the Pesapal Membership folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Once the plugin is activated there will be an admin section (Membership) where you configure your PesaPal credentials and the registration fee

== Frequently Asked Questions ==

= Why am I not getting payment confirmation emails? =

Make sure you have configured the Callback IPN URL in your PesaPal merchant account as the URL in the Plugin Settings page

= Why cant users register? =

Make sure you have turned on User Registration in your WordPress settings
